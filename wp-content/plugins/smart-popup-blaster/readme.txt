Easy to use popup plugin, highly customizable, scroll, click & time-delay triggered popups. Add forms, social media boxes, videos & more.

== Description ==

Smart PopUp Blaster plugin can display multiple popups, it's highly customizable so you can take control over your popup behaviour. Add call to actions, increase your social followers, increase your mailing lists by adding any type of forms or display any message in a simple popup.
With this fully-responsive plugin, your pop-ups will look fantastic on every device.

= Features: =

* Multiple PopUps
* Choose where to show each popup
* Exit Intent popup trigger
* Time Delay popup trigger
* Scroll popup trigger
* Click popup trigger
* Fully-Responsive
* Highly Customizable
* Full control over popup content
* Many Animation Effects
* Mailchimp is supported
* Display any shortcode in popup



== Installation ==

1. Upload `smart-popup-blaster` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Smart PopUp Blasetr in admin panel > Add New PopUP
4. Enjoy
